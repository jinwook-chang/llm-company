# Samsung System LSI 및 Image Sensor 공개 요약

## System LSI Business

Samsung System LSI Business는 SoC와 image sensor 등 시스템 반도체를 담당한다. 2025년 4분기에는 seasonal demand softness와 key customer product schedule adjustment로 earnings가 둔화되었다.

## Image Sensor

같은 발표에서 Samsung은 new 200MP and 50MP big-pixel products에 힘입어 image sensor revenue가 성장했다고 밝혔다. 2026년에는 expanded 200MP lineup을 통해 image sensor leadership을 강화할 계획이라고 설명했다.

## Nanoprism과 Fine-pixel

Samsung은 image sensor 사업에서 fine-pixel technology와 Nanoprism technology를 강조한다. Nanoprism은 light sensitivity를 향상시키는 기술로 소개되며, sensor competitiveness 강화를 위한 요소로 언급되었다.
